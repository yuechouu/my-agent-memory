## 短期计划（1-2 周）

### 1. 热层记忆注入策略 [P0]

**问题**：当前热层注入所有记忆类型，coding agent 会收到无关信息。

**方案**：
- 始终注入：`user-identity`, `user-preference`, `feedback-correction`
- 条件注入：根据 Agent 类型决定
- 从不注入：`user-context`, `feedback-confirmation`

**实现**：
1. 更新 `memory_types.py` 添加 `inject_policy` 字段
2. 更新 `hot_layer.py` 的过滤逻辑
3. 添加配置文件支持

### 2. 学习功能清理 [P2]

**问题**：patrol 中的学习逻辑已移除，但相关代码（_phase2_learning 等）还在。

**方案**：
- 删除 patrol.py 中的学习相关方法
- 删除 CLI 中的 --learn 参数
- 删除 MCP 工具中的 include_learning 参数
- 保留 learn 命令（由 Agent 调用，不是 patrol）

---

## 中期计划（1-2 月）

### 3. 记忆质量评估

**目标**：评估记忆的质量，优先展示高质量内容。

**指标**：
- 完整性：内容是否完整
- 准确性：是否有错误
- 时效性：是否过期
- 引用：是否有有效引用

### 4. 记忆关联图谱

**目标**：建立记忆之间的关联关系。

**功能**：
- 自动检测相关记忆
- 可视化记忆关联
- 基于关联推荐相关记忆

### 5. 多模态记忆

**目标**：支持图片、音频、视频等多模态内容。

**功能**：
- 图片记忆（OCR + 向量化）
- 音频记忆（转录 + 向量化）
- 视频记忆（关键帧提取）

---

## 长期计划（3-6 月）

### 6. 分布式记忆

**目标**：支持多设备、多 Agent 的记忆同步。

**功能**：
- 记忆同步协议
- 冲突自动解决
- 离线支持

### 7. 记忆压缩

**目标**：自动压缩冗余记忆，节省存储空间。

**功能**：
- 相似记忆合并
- 过期记忆归档
- 智能摘要生成

### 8. 记忆推理

**目标**：基于记忆进行推理和决策。

**功能**：
- 因果推理
- 时间线推理
- 因果链分析
